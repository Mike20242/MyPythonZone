About DevBlog

Welcome to DevBlog! This is a personal blog mini-project built with Python and Flask.

Our goal is to demonstrate how to create beautiful, dynamic web applications with minimal effort using modern design principles. This project leverages an aesthetic dark mode layout styled completely with Vanilla CSS.

Technologies Used

  • Backend: Python & Flask
  • Frontend: HTML5, Vanilla CSS, Jinja2 Templates
  • Design System: Custom dark theme with glassmorphism and smooth micro-animations.
